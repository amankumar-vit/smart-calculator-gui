# 🖥️ Smart Calculator GUI

A modern calculator built using Python and Tkinter with a clean UI.

## 🚀 Features
- Basic calculator (+, -, *, /)
- Modern dark UI
- Button-based input (like real calculator)
- Error handling

## 🛠️ Tech Used
- Python
- Tkinter

## ▶️ How to Run

1. Install Python
2. Run the command:
   python main.py
